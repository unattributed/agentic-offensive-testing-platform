# Licensing readiness

AOTP is public source-available work with all rights reserved. No open-source license is granted by the current `LICENSE.md`.

Public repository visibility is for review, portfolio, and evaluation. It does not authorize copying, modification, redistribution, sublicensing, operation, or use outside the repository owner’s authorized development and evaluation activities unless a separate written license agreement is provided.

Contributions must have clear provenance. No code from reference projects is vendored. Optional integrations use clean-room contracts and external process boundaries.

Before any broader licensing or distribution change, review every dependency and transitive dependency, contributor rights, generated assets, documentation sources, trademarks, data handling, support obligations, export considerations, public-risk posture, and warranty and liability language. Keep private evidence, profiles, scopes, reports, and campaign memory outside every distribution.

The [evaluator license model](evaluator-license-model.md) is a non-binding engineering draft marked
for legal review. It grants no rights and cannot be used until counsel and both parties approve a
final agreement.
