# Commercialization plan

Potential paths are continued private use, commercial licensing, limited evaluator licensing, managed client delivery, or a later open-core release. No path is selected by this document.

Readiness work includes provenance review, dependency license inventory, contributor assignments, trademark review, security support terms, privacy and retention terms, export and sanctions review where applicable, warranty and liability language, pricing, and a public-release redaction audit. Legal counsel should review any external license.
