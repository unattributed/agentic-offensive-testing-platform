# Candidate evaluation workflow

An evaluator provisions an isolated environment and approved accounts, provides written authorization and rules of engagement, and defines permitted categories and evidence handling. The candidate creates private alias-based profile and scope files, plans a campaign, obtains approval for risky steps, and captures reproducible evidence.

Review should assess policy discipline, WSTG coverage, evidence quality, handling of uncertainty, safe account use, and report accuracy. It should not reward target expansion or destructive behavior.
