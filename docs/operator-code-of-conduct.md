# Operator code of conduct

Operators must:

- obtain and respect written authority;
- treat scope as a hard boundary;
- minimize requests and sensitive data;
- use provisioned accounts only as permitted;
- stop on instability, lockout risk, ambiguity, or sponsor request;
- protect evidence and confidentiality;
- report accurately without exaggeration;
- avoid duplicate noise and public disclosure; and
- preserve a clear human decision trail.

Commercial incentive, evaluation pressure, or model output never justifies violating these duties.
