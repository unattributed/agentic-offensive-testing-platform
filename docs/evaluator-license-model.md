# Evaluator license model

A future evaluator license may grant a named evaluator limited, non-transferable access for a defined review window and purpose. It should prohibit redistribution, reverse engineering where lawful, production use, target use outside written scope, and disclosure of private materials.

The agreement should address confidentiality, feedback rights, data retention, security incidents, support, termination, warranty, liability, governing law, and return or deletion. It requires legal review before use.
