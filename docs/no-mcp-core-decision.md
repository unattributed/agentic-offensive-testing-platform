# No MCP Core Decision

MCP is not part of the core AOTP architecture. The Deep Agent receives campaign-governed native
AOTP tools directly so scope, risk, budgets, approvals, evidence classification, and audit records
remain inside one deterministic control plane.

No MCP dependency should be added in the current roadmap. MCP may be reconsidered later only as an
optional adapter after the native governance model is implemented and proven. Such an adapter
would not replace or bypass the native registry, policy gate, human steering, or evidence rules.
